# Портфолио Макса Водяного

Современное портфолио дизайнера, созданное на чистом HTML, CSS и JavaScript.

## Структура проекта

```
├── index.html      - основной файл с разметкой
├── styles.css      - стили оформления
├── script.js       - интерактивность и функционал
└── README.md       - этот файл
```

## Возможности

✅ Отзывчивый дизайн (адаптируется под мобильные, планшеты, десктоп)
✅ Герой секция с привлекающим взглядом
✅ Раздел "Обо мне" с навыками
✅ Галерея портфолио с эффектами при наведении
✅ Форма контакта
✅ Гладкая прокрутка между разделами
✅ Анимации при загрузке

## Как использовать

### 1. Быстрый старт
- Просто откройте файл `index.html` в браузере
- Сайт полностью функционален без сервера

### 2. Кастомизация контента

#### Изменение имени и информации
Отредактируйте в `index.html`:
- Замените "Макс Водяной" на ваше имя
- Обновите email и телефон в разделе kontakty


#### Добавление проектов в портфолио
В секции portfolio найдите `.portfolio-item` и добавьте новые карточки:

```html
<div class="portfolio-item">
    <div class="portfolio-image">
        <img src="путь-к-вашей-картинке" alt="Название проекта">
    </div>
    <div class="portfolio-info">
        <h3>Название проекта</h3>
        <p>Описание</p>
    </div>
</div>
```

#### Загрузка своих изображений
1. Создайте папку `images/` в директории проекта
2. Поместите ваши картинки туда
3. Обновите пути в HTML: `src="images/ваша-картинка.jpg"`

### 3. Настройка цветов

В файле `styles.css` найдите переменные CSS:

```css
:root {
    --primary-color: #1a1a1a;      /* Основной черный */
    --secondary-color: #f0f0f0;    /* Фоновый серый */
    --accent-color: #ff6b35;       /* Оранжевый акцент */
    --text-color: #333;            /* Основной текст */
    --light-text: #666;            /* Светлый текст */
}
```

Измените значения на нужные вам цвета в формате HEX.

### 4. Отправка форм

Текущая форма показывает алерт. Для реальной отправки email:

**Вариант 1: Через FormSubmit (бесплатный сервис)**
1. Откройте https://formspree.io/
2. Создайте форму и получите ID
3. Обновите action в форме:

```html
<form action="https://formspree.io/f/ВАШ_ID" method="POST">
```

**Вариант 2: Через Telegram бот**
Используйте TelegramBot API для отправки форм в Telegram.

## Заливка на хостинг

### Для бесплатной заливки:
1. **Netlify**: Перетащите папку проекта на https://netlify.com
2. **Vercel**: Загрузите на https://vercel.com
3. **GitHub Pages**: Загрузите репозиторий в GitHub и включите Pages

### Для вашего домена:
1. Купите домен у регистратора (RuCenter, REG.RU и т.д.)
2. Загрузите файлы через FTP на хостинг
3. Настройте DNS записи

## Оптимизация SEO

Обновите в `index.html` в разделе `<head>`:

```html
<meta name="description" content="Портфолио дизайнера Макса - создание логотипов и дизайна">
<meta name="keywords" content="дизайн, логотип, портфолио, макс">
<meta name="author" content="Макс Водяной">
```
